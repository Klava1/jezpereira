## Image assets for jezpereira.com

Drop image files in this folder and reference them from the HTML.

Suggested filenames (currently referenced or recommended):

- og-cover.jpg          — social-share image (1200 × 630 px), used in <meta og:image>
- portrait-hero.jpg     — main portrait, used on homepage hero (recommended 1600px wide)
- portrait-story.jpg    — second portrait for The Story page
- show-koko.jpg         — image for the KOKO 2025 highlight
- group-syck.jpg        — group/agency photo for The Family page

You can use any filenames — just update the `src` attribute in the HTML to match.

The site currently runs without images using gradient placeholders, so it'll look complete the moment you drop files in.
