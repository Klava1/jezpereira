  H1 (industrial display caps, +0.09em tracking, with stage-lamp halo):

      Animation choreography on the H1 letters (129 BPM = "Lost in Musyck" tempo):
        idle    →  entrance fadeUp (parent), no per-letter motion
        unison  →  every letter pulses together, +3 %        (2 bars)
        wave    →  pulse cascades letter-by-letter           (2 bars)
        bold    →  +7 % pulse with accent text-shadow glow   (4 bars / "the drop")
        ↑ unison → wave → bold repeats THREE TIMES
        fade    →  beat-by-beat decay 7 % → 2.2 %            (2 bars)
        subtle  →  quarter-time resting heartbeat at 2.2 %   (infinite)

  Scroll hint (bottom, mono caps, gentle bob):
      SCROLL

----------------------------------------------------------------

  Overline:  Movement One · 1993 — 2001
  H2:        Bombay to London

  Body:
      Jez Pereira is a globally recognised DJ and producer originally
      from Bombay, India, where his musical journey began in 1993 at
      the city's legendary venues — including Piano Bar and China
      Garden.

      In 2001, he moved to London and immersed himself in the UK
      underground dance scene, drawing inspiration from iconic clubs
      such as KOKO and Ministry of Sound, where his distinctive
      *freestyle* sound and musical identity evolved.

----------------------------------------------------------------

----------------------------------------------------------------

  Overline:  Movement Two · 2001 — 2015
  H2:        The World

  Body:

      and Pacha Buzios, Brazil, to performances at Armani Privé Dubai

================================================================
EDITING GUIDE — where copy lives
================================================================

  GLOBAL CHROME
    Nav links / labels ............ /components/Nav.tsx (LINKS array)
    Footer columns ................ /components/Footer.tsx
    Music player labels ........... /components/MusicPlayer.tsx (LABEL, TRACK)

  HOME
    Hero overline + title ......... /components/Hero.tsx
    "Currently Playing" + doors ... /app/page.tsx

  STORY
    Page header copy .............. /app/story/page.tsx
    Each movement (1 file each) ... /content/bio/0X-*.mdx

  MUSIC
    Page header copy .............. /app/music/page.tsx
    Mix-platform links ............ /app/music/page.tsx (MIX_LINKS array)
    Each release (1 file each) .... /content/releases/*.mdx

  LIVE
    Page header copy .............. /app/live/page.tsx
    Bookings card ................. /app/live/page.tsx (bottom)
    Each gig (1 file each) ........ /content/gigs/*.mdx
        upcoming: true            → moves into the "Calendar" section
        featured: true            → optional flag (for future styling)

  FAMILY
    Page header + intro + closing . /app/family/page.tsx
    Each of the four arm cards .... /app/family/page.tsx

  Conventions:
    *single asterisks* in MDX render as accent-pink italic
    Each gig / release / bio file's frontmatter (between the ---
       lines) drives the rendered fields directly.
